Track your customers the easy way with Customer.io

== Description ==

This extension allows users to sign up on your Customer.io email marketing account and automatically tracks financial and purchase details allowing you fine-grained control over your mailing lists.

== Installation ==

1. Unzip the downloaded zip file
2. Upload the included folder to '/wp-content/plugins' directory of your WordPress installation
3. Activate the plugin via the WordPress Plugins page
